# Página Web del Alfarero

Esta es una página web básica para un alfarero, creada con HTML, CSS y JavaScript.

## Uso

1. Abre `index.html` en tu navegador web para ver la página.
2. Personaliza el contenido en `index.html` según tus necesidades.
3. Modifica los estilos en `styles.css` para cambiar la apariencia.
4. Agrega funcionalidades en `script.js`.

## Estructura de Archivos

- `index.html`: Página principal
- `styles.css`: Estilos CSS
- `script.js`: JavaScript básico

## Troubleshooting

- Si la página no se carga, asegúrate de que todos los archivos estén en el mismo directorio.
- Para ver cambios en CSS o JS, refresca la página en el navegador.
- Si hay problemas con el navegador, intenta con otro navegador moderno.
